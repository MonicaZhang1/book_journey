FAQ

1. What is the purpose of this application?

This application was created to help users easily track all the books they've read and their thoughts on the books all in one place. This makes it easy for users to reference the books if they want to reread it or to give recommendations.

2. How should I use the application?

The main feature of this application is the Home page that shows all the books that are read. If a book isn't listed, use the Add button to add a new book with its information. If there are any mistakes, the book information can be edited/deleted with the Edit and Delete buttons.

3. How can I access reviews?

There are 2 ways to view reviews. The first option is to click the Review tab to go to the page that shows all the reviews. The second option, on the Home page click the book card and it will redirect to the review of the specific book.
